# React Eurosport test

A very simple test apps.

### Installing

```
npm install / yarn
```

#### Development

```
npm start / yarn start
```

#### Test

```
npm run test / yarn test
```

# Dependencies

#### ReactJs

```
 JavaScript library[3] for building user interfaces
```

#### Webpack

```
 is a static module bundler for modern JavaScript applications
```

#### Typescript

```
 is designed for development of large applications and transcompiles to JavaScript
```

#### Material UI

```
 Framework CSS and HTML
```

#### Axios

```
 Promise based HTTP client for the browser and node.js
```

#### Jest

```
 is testing Framework with a focus on simplicity
```

#### Enzyme

```
 is testing utility for React that makes it easier to test your React Components' output.
```
